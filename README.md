# AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes against job descriptions using NLP and Large Language Models.

## Features

- ATS Score Calculation
- Resume Skill Extraction
- Job Description Matching
- Missing Skill Detection
- AI-Based Resume Suggestions
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- NLP
- OpenAI API
- Scikit-Learn

## Project Status

Currently under development.
